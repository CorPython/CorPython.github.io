Hugo提交博客及推送到github的流程

2019年11月7日
| 阅读

在博客根目录下运行下面命令,并在post目录下生成markdown博客文件

hugo new post/xxxx.md

在博客根目录下运行下面命令

hugo //将本地文件重新编译
hugo server//在本地生成

注意,可以在本地查看博客变化,等自己觉得合适了就可以使用下面命令提交到github

在hugo本地的public目录下使用如下命令

git add -A //将修改的文件提交到缓冲区
git commit -m '提交说明'
git push -u origin master //推送到远端仓库主分支上
