C++编译错误

2020年1月7日
| c++
|70 words
| 阅读
 | 阅读时间大约1分钟
文章目录

编译cpp文件时报错:

ld.exe: can not open file ...xxx.exe:permission denied collect2.exe error 
ld returned 1 exit status

原因:代码当前目录（包括子目录）下有同名的.exe文件，会报此错误，只要把名字改一下就行了
