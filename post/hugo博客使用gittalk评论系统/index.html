Hugo博客使用gittalk评论系统

2019年11月11日
| 阅读

1.进入github官方网站,登陆自己的github->settings->Devloper settings->Github Apps会出现下图

  • Application name:给应用起个名字,根据自己的想法
  • Homepage:写你的博客地址
  • Application description:为你的应用写一段描述
  • Application callback URL:回调地址,可以写你的博客地址

2.点击Register application创建,生成自己的client key和client Secret

3.找到配置评论的文件,比如我的文件是comments.html,该文件的位置在\themes\maupassant\layouts\partials\comment.html

{{ if .Site.Params.enableGitalk }}
<div id="gitalk-container"></div>
<link rel="stylesheet" href="https://unpkg.com/gitalk/dist/gitalk.css">
<script src="https://unpkg.com/gitalk/dist/gitalk.min.js"></script>
<script>
  const gitalk = new Gitalk({
    clientID: '{{ .Site.Params.Gitalk.clientID }}',
    clientSecret: '{{ .Site.Params.Gitalk.clientSecret }}',
    repo: '{{ .Site.Params.Gitalk.repo }}',
    owner: '{{ .Site.Params.Gitalk.owner }}',
    admin: ['{{ .Site.Params.Gitalk.owner }}'],
    id: decodeURI(location.pathname), // Ensure uniqueness and length less than 50
    distractionFreeMode: false // Facebook-like distraction free mode
  });
  (function() {
    if (["localhost", "127.0.0.1"].indexOf(window.location.hostname) != -1) {
      document.getElementById('gitalk-container').innerHTML = 'Gitalk comments not available by default when the website is previewed locally.';
      return;
    }
    gitalk.render('gitalk-container');
  })();
</script>
{{ end }}

注意:使用hugo server在本地启动看不到,如下图,另外记住绑定自己的仓库,一般托管到git page上的都默认绑定了你博客所在仓库

4.所有操作完成后,使用git命令推送到github上.最终,可以看到评论已经上线了
